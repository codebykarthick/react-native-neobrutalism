# Breadcrumb

A navigation component that displays the current page location within a hierarchical structure.

## Import

```tsx
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
  BreadcrumbSeparator,
} from 'react-native-neobrutalism';
```

## Basic Usage

```tsx
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink onPress={() => navigate('/')}>Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink onPress={() => navigate('/products')}>Products</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current Page</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## Component Structure

The Breadcrumb component uses a composable structure:

| Component | Purpose |
|-----------|---------|
| `Breadcrumb` | Root wrapper that provides context |
| `BreadcrumbList` | Flex container for items (horizontal row) |
| `BreadcrumbItem` | Wrapper for individual breadcrumb entries |
| `BreadcrumbLink` | Clickable navigation link |
| `BreadcrumbPage` | Current page indicator (non-clickable) |
| `BreadcrumbSeparator` | Visual divider between items |

## Props

### Breadcrumb

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | **required** | BreadcrumbList content |
| `style` | `ViewStyle` | - | Container styles |
| `themeOverride` | `NeobrutalismThemeOverride` | - | Theme overrides for all children |
| `accessibilityLabel` | `string` | `'Breadcrumb'` | Accessibility label |

### BreadcrumbList

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | **required** | BreadcrumbItems and BreadcrumbSeparators |
| `style` | `ViewStyle` | - | Container styles |

### BreadcrumbItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | **required** | BreadcrumbLink or BreadcrumbPage |
| `style` | `ViewStyle` | - | Container styles |

### BreadcrumbLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | **required** | Link text or element |
| `onPress` | `() => void` | - | Press handler for navigation |
| `disabled` | `boolean` | `false` | Disable the link |
| `style` | `ViewStyle` | - | Container styles |
| `textStyle` | `TextStyle` | - | Text styles |

### BreadcrumbPage

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | **required** | Page text or element |
| `style` | `ViewStyle` | - | Container styles |
| `textStyle` | `TextStyle` | - | Text styles |

### BreadcrumbSeparator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | `'›'` | Custom separator content |
| `style` | `ViewStyle` | - | Container styles |

## Accessibility

The Breadcrumb component includes built-in accessibility support:

- `accessibilityRole="link"` on BreadcrumbLink
- `accessibilityState={{ disabled }}` reflects disabled state
- `aria-current="page"` on BreadcrumbPage
- `aria-hidden` on separators (decorative)

```tsx
<BreadcrumbLink
  onPress={() => navigate('/home')}
  accessibilityLabel="Navigate to home page"
>
  Home
</BreadcrumbLink>
```

## Usage with Navigation

Integrate with React Navigation or other navigation libraries:

```tsx
import { useNavigation } from '@react-navigation/native';

function MyBreadcrumb() {
  const navigation = useNavigation();

  return (
    <Breadcrumb>
      <BreadcrumbList>
        <BreadcrumbItem>
          <BreadcrumbLink onPress={() => navigation.navigate('Home')}>
            Home
          </BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
          <BreadcrumbLink onPress={() => navigation.navigate('Products')}>
            Products
          </BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
          <BreadcrumbPage>Product Details</BreadcrumbPage>
        </BreadcrumbItem>
      </BreadcrumbList>
    </Breadcrumb>
  );
}
```
