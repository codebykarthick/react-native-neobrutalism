# Button

A neobrutalism-styled button component with multiple variants, sizes, and press animations.

## Import

```tsx
import { Button } from 'react-native-neobrutalism';
```

## Basic Usage

```tsx
<Button label="Click me" onPress={() => console.log('Pressed!')} />
```

<Canvas of={ButtonStories.Default} />

## Variants

Six visual variants are available:

```tsx
<Button label="Warning" variant="warning" />
<Button label="Danger" variant="danger" />
<Button label="Success" variant="success" />
<Button label="Outlined" variant="outlined" />
```

## Custom Styling

Override styles at the component level:

```tsx
<Button
  label="Custom"
  style={{ backgroundColor: 'purple' }}
  textStyle={{ color: 'white', fontWeight: 'bold' }}
  shadowStyle={{ backgroundColor: 'darkpurple' }}
/>
```

### Disable Shadow

Pass `shadowStyle={null}` to remove the shadow:

```tsx
<Button label="No Shadow" shadowStyle={null} />
```

### Disable Animation

```tsx
<Button label="No Animation" disableAnimation />
```

<Canvas of={ButtonStories.NoAnimation} />

## Component-Level Theme Override

Override theme values for a specific button:

```tsx
<Button
  label="Custom Theme"
  themeOverride={{
    colors: { primary: '#FF0000' },
    shadow: { offsetX: 8, offsetY: 8 },
  }}
/>
```

## Accessibility

The Button component includes built-in accessibility support:

- `accessibilityRole="button"` is set automatically
- `accessibilityLabel` defaults to the `label` prop
- `accessibilityState={{ disabled }}` reflects the disabled state

```tsx
<Button
  label="Submit"
  accessibilityLabel="Submit the form"
/>
```
