# Card

A neobrutalism-styled card container component for displaying grouped content with header, content, and footer sections.

## Import

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from 'react-native-neobrutalism';
```

## Basic Usage

```tsx
<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
  </CardHeader>
  <CardContent>
    <Text>Card content goes here.</Text>
  </CardContent>
  <CardFooter>
    <Button label="Cancel" variant="secondary" />
    <Button label="Save" />
  </CardFooter>
</Card>
```

## Sub-Components

The Card component uses a compound component pattern with the following sub-components:

| Component | Description |
|-----------|-------------|
| `Card` | Main container with shadow and border |
| `CardHeader` | Header section with padding |
| `CardTitle` | Bold title text |
| `CardDescription` | Muted description text |
| `CardContent` | Main content area with padding |
| `CardFooter` | Footer section for actions (flex row) |

## Custom Theme

Override theme values at the component level:

<Canvas of={CardStories.CustomTheme} />

```tsx
<Card
  themeOverride={{
    shadow: { offsetX: 6, offsetY: 6 },
    border: { radius: 12 },
  }}
>
  <CardHeader>
    <CardTitle>Custom Styled</CardTitle>
  </CardHeader>
  <CardContent>
    <Text>Uses custom shadow and border radius.</Text>
  </CardContent>
</Card>
```

## Without Shadow

Pass `shadowStyle={null}` to remove the shadow:

```tsx
<Card shadowStyle={null}>
  <CardHeader>
    <CardTitle>No Shadow Card</CardTitle>
  </CardHeader>
  <CardContent>
    <Text>This card has no shadow effect.</Text>
  </CardContent>
</Card>
```

## Props

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Card content (sub-components) |
| `style` | `ViewStyle` | - | Override container styles |
| `shadowStyle` | `ViewStyle \| null` | - | Override shadow styles, `null` to hide |
| `themeOverride` | `NeobrutalismThemeOverride` | - | Component-level theme overrides |
| `accessibilityLabel` | `string` | - | Accessibility label |

### CardHeader

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Header content |
| `style` | `ViewStyle` | Override container styles |

### CardTitle

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Title text |
| `style` | `TextStyle` | Override text styles |

### CardDescription

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Description text |
| `style` | `TextStyle` | Override text styles |

### CardContent

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Main content |
| `style` | `ViewStyle` | Override container styles |

### CardFooter

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Footer content (typically buttons) |
| `style` | `ViewStyle` | Override container styles |

## Accessibility

- Use `accessibilityLabel` on Card to describe the card's purpose
- CardTitle and CardDescription provide semantic content for screen readers
- Consider using proper heading levels within cards for navigation
